<objective>
Extract all historical sync state and logic out of GooseBLEClient into a dedicated `GooseBLEHistoricalManager` final class (callback pattern, not @Observable, not Combine — matching the GooseBLEBondingManager precedent). GooseBLEClient retains a private `historicalManager` instance and exposes proxy computed properties (`isHistoricalSyncing`, `historicalSyncStatus`, `historicalSyncRunID`) that forward to the manager, so all existing read call sites across GooseBLEClient+*.swift compile and behave unchanged. All write sites delegate to manager methods.

Purpose: Decouple a large, cohesive subsystem (~30 vars) from the GooseBLEClient god-object so it can later be tested independently (ARCH-01, Phase 72). Implements BLE5-03.

Output: New `GooseBLEHistoricalManager.swift`; modified `GooseBLEClient.swift` and the four historical-touching extension files; pbxproj reference for the new file.
</objective>
