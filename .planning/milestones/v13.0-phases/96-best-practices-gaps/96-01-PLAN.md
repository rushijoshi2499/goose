    Replace the `_ = try? bridge.request(method: "capture.import_frame_batch", ...)` block with a do/catch.     GooseAppModel+Upload.swift line 251 — sync.backfill_streams:

    The current pattern is:
      guard let report = try? localRust.request(method: "storage.compact_raw_evidence", ...) else { return }
    Replace with:
      let report: [String: Any]
      do {
        guard let r = try localRust.request(method: "storage.compact_raw_evidence", ...) else { return }
