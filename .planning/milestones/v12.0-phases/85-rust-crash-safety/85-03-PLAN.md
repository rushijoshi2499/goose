clippy fires on), convert the 8 test-code `.unwrap()` to `.expect()` (D-03), then remove the
`#![allow(clippy::unwrap_used)]` shield Plan 1 added.

  `timestamps_aligned` guard already proves Some. Use `.expect("...")` preserving the existing
  safety reasoning (Pattern 3).

  `a.total_cmp(b)` (Pattern 4 / Pitfall 4). `finite` is pre-filtered to `.is_finite()`;
  `f64::total_cmp` returns `Ordering` directly with no Option and handles NaN deterministically.
- Line ~3609 `trend_score.unwrap()` inside an `if trend_score.is_some()` block — refactor to
  `if let Some(ts) = trend_score` and use `ts` (Pattern 2), eliminating the unwrap entirely.

Purpose: Close 3 of the 5 real production crash-safety holes and shrink the allow list.
Output: metrics.rs with no `.unwrap()`, no allow shield, passing the deny lint.
