<objective>
Create GooseStrainAccumulator — a Swift actor that accumulates Edwards Zone Load strain in real time
from HR samples delivered via the existing ble.onLiveHeartRate callback, throttles publication to
≤1 update/3s, and publishes the running total to GooseAppModel.liveWorkoutStrain (@Published Double).
Accumulation is guarded by activeActivityPersistence — no strain during rest. Strain resets on
workout begin and is frozen+zeroed on workout end.

Purpose: DATA-02 — live strain tile on the workout screen updates during active sessions without
any Rust bridge latency (computation is Swift-side only).

Output: GooseStrainAccumulator.swift (new), GooseAppModel.swift + GooseAppModel+ActivityRecording.swift
(edited). No new dependencies.
</objective>

  Declare `actor GooseStrainAccumulator` with these stored properties (all automatically actor-isolated):
  - accumulatedLoad: Double = 0
  - lastSampleDate: Date? = nil
  - lastPublishedAt: Date = .distantPast
  - isFrozen: Bool = false
  - maxHR: Double = GooseStrainAccumulator.defaultMaxHR

  Static constants (accessible without actor isolation — use `nonisolated static let`):
  - defaultMaxHR: Double = 190
  - publishInterval: TimeInterval = 3
  - maxSampleGap: TimeInterval = 30   // samples with gap > 30s are stale; discard and reset anchor

  Methods:
  - func ingest(bpm: Int, date: Date): guard !isFrozen, then if lastSampleDate is nil set it and
    return. Compute interval = date.timeIntervalSince(last); guard interval > 0, interval <
    GooseStrainAccumulator.maxSampleGap — if gap violated, update lastSampleDate and return.
    Compute hrPct = Double(bpm) / maxHR; switch on hrPct: <0.60→1.0, 0.60..<0.70→2.0,
    0.70..<0.80→3.0, 0.80..<0.90→4.0, default→5.0. Accumulate:
    accumulatedLoad += multiplier * interval. Set lastSampleDate = date.

  - func pollIfReady(now: Date) -> Double?: guard now.timeIntervalSince(lastPublishedAt) >=
    GooseStrainAccumulator.publishInterval; set lastPublishedAt = now; return accumulatedLoad.
    Returns nil if throttle window has not elapsed.

  - func reset(): accumulatedLoad = 0; lastSampleDate = nil; lastPublishedAt = .distantPast;
    isFrozen = false.

  - func freeze(): isFrozen = true. (Called on workout end; ingest becomes a no-op.)

  - func setMaxHR(_ bpm: Double): maxHR = bpm. (Hook for Phase 72 user profile HRmax.)

  Note: HR samples from onLiveHeartRate are already sanitized by GooseHRSanitizer (25-220 BPM) —
  no second range check needed inside ingest.
