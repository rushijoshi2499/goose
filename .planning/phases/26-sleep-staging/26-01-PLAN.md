<objective>
Implement the actigraphy spine for sleep staging (ALG-SLP-03): a new `sleep_staging.rs`
module that aggregates gravity-table rows into 1-minute epochs, computes per-epoch activity
counts, applies the Cole-Kripke (1992) 7-term weighted D score, and classifies each epoch as
wake or sleep. Expose it through a new `metrics.sleep_staging` bridge method.

Purpose: WHOOP's staging is server-side (confirmed by Ghidra, see 26-CONTEXT.md), so there are
no proprietary coefficients to extract. The published Cole-Kripke algorithm with a mandatory
`actigraphy_uncalibrated` quality flag and a configurable scale factor is the correct, honest
approach. This plan delivers the binary wake/sleep spine; Plan 26-02 extends it to 4 classes.

Output: `sleep_staging.rs` module (types + classifier + named constants), `lib.rs` module
registration, and the `metrics.sleep_staging` bridge dispatch + wrapper.
</objective>
